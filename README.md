# Counter App

## Descrizione
Questa è una semplice applicazione web sviluppata in **JavaScript puro** che permette all’utente di incrementare, decrementare e resettare un contatore. Il valore del contatore viene salvato nel **Local Storage**, così da essere persistente tra un reload della pagina e l’altro.

Il progetto è pensato come esercizio didattico per consolidare concetti base di:

- Manipolazione del DOM
- Event handling con **event delegation**
- Funzioni di callback
- CSS moderno con **Flexbox** e **Grid**
- Local Storage

## Funzionalità

- Visualizzazione del contatore, inizializzato a 0 o al valore salvato
- Incremento e decremento tramite pulsanti “+” e “−”
- Pulsante **Reset** per riportare il contatore a 0
- Disabilitazione automatica del pulsante decremento quando il contatore è 0
- Persistenza del valore tramite Local Storage

## Tecnologie utilizzate

- HTML5
- CSS3 (Flexbox e Grid)
- JavaScript ES6+
- Local Storage

## Come usare

1. Clonare il repository:
git clone https://github.com/MarcoSar1991/counter_project.git

2. Aprire index.html in un browser moderno.

3. Interagire con i pulsanti per incrementare, decrementare o resettare il contatore.
